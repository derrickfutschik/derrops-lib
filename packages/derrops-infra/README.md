# Derrops Infrastructure

This package contains the infrastructure code for the Derrops platform. It is used to deploy the Derrops platform to AWS.
It utilizes the AWS CDK to deploy the infrastructure.
It is a TypeScript package that is used to deploy the Derrops platform to AWS.
The infrastructure layer is does not support feature deploys unlike the amplify deploy of the derrops-ports app or the lambda deploy of the derrops-cloud app.
