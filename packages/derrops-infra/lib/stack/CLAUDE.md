Each file in here contains a CloudFormation Stack written by the CDK, and a corresponding ${stack-name}.md file that describes the stack and its purpose, as well as a mermaid infra diagram.
When working with these stacks, you should:

- Read the ${stack-name}.md file to understand the stack and its purpose.
- View the mermaid infra diagram to understand the stack and its components.
- Make changes to the stack in the ${stack-name}.ts file.
- Update the ${stack-name}.md file to reflect the changes.
- Update the mermaid infra diagram to reflect the changes.
