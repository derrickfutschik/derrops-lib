# Security Overview

This section provides security and compliance information for customers and prospective customers evaluating the SLAOps platform.

## Topics

- **Architecture security** — how data flows through the platform and where it is protected
- **Authentication & authorisation** — how users and services authenticate
- **Data handling** — what data is collected, where it is stored, and how long it is retained
- **Encryption** — encryption at rest and in transit
- **Compliance** — certifications and regulatory considerations
