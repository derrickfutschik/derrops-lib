# DevOps User Stories

| ID          | Done | Description                                                                                                                                |
| ----------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **OPS-001** | ☐    | As DevOps I want to separate infrastructure folder from the backend folder                                                                 |
| **OPS-002** | ☐    | As DevOps I want backend folder to contain only the amplify/cdk code needed for the lambda function                                        |
| **OPS-003** | ☐    | As DevOps I want infrastructure folder to contain the code needed for various stacks (api gateway, rds, cognito, etc)                      |
| **OPS-004** | ☐    | As DevOps I want to Deploy requester to AWS                                                                                                |
| **OPS-005** | ☐    | As DevOps I want to Deploy requester to local                                                                                              |
| **OPS-006** | ☐    | As DevOps I want API Gateway to Be defined in OpenAPI and CDK (backend module)                                                             |
| **OPS-007** | ☐    | As DevOps I want API Gateway to Have Cognito auth configured                                                                               |
| **OPS-008** | ☐    | As DevOps I want API Gateway to Support features deploys by selecting different lambdas based off of the header                            |
| **OPS-009** | ☐    | As DevOps I want RDS Defined by CDK (backend module)                                                                                       |
| **OPS-010** | ☐    | As DevOps I want RDS to be Serverless                                                                                                      |
| **OPS-011** | ☐    | As DevOps I want RDS to have IAM Auth                                                                                                      |
| **OPS-012** | ☐    | As DevOps I want RDS to support Feature Deploys per Schema                                                                                 |
| **OPS-013** | ☑   | As DevOps I want Sign in with Email configured                                                                                             |
| **OPS-014** | ☐    | As DevOps I want Sign in with Google configured                                                                                            |
| **OPS-015** | ☐    | As DevOps I want Local Proxy Landing page for all services that are used with the location where they are setup and links to documentation |
| **OPS-016** | ☐    | As DevOps I want Local Proxy setup to switch between AWS and local environments for each service                                           |
