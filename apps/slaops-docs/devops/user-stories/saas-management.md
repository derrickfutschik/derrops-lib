# SAAS Management User Stories

| ID | Done | Description |
|---|---|---|
| **SAAS-001** | ☐ | As an Admin I want to be able to create a new SAAS application and save it in the SQL database |
| **SAAS-002** | ☐ | As an Admin I want to be able to edit a SAAS application and save it in the SQL database |
| **SAAS-003** | ☐ | As an Admin I want to be able to delete a SAAS application and remove it from the SQL database |
| **SAAS-004** | ☐ | As an Admin I want to be able to view a SAAS application and see the details in the SQL database |
