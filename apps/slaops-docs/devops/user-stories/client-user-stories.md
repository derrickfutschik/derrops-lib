# Requester User Stories

| ID | Done | Description |
|---|---|---|
| **REQ-001** | ☑ | As a Requester I want to sign in with AWS Cognito |
| **REQ-002** | ☑ | As a Requester I want to sign out with AWS Cognito |
| **REQ-003** | ☐ | As a Requester I want to search for an OpenAPI specification by name or description |
| **REQ-004** | ☐ | As a Requester I want to search for an OpenAPI specification by what SAAS products I observe |
| **REQ-005** | ☑ | As a Requester I want to see a list of all the OpenAPI specifications |
| **REQ-006** | ☑ | As a Requester I want to see the details of an OpenAPI specification |
| **REQ-007** | ☑ | As a Requester I want to add an OpenAPI specification |
| **REQ-008** | ☐ | As a Requester I want to edit an OpenAPI specification |
| **REQ-009** | ☐ | As a Requester I want to delete an OpenAPI specification |
| **REQ-010** | ☑ | As a Requester I want to make requests according to the OpenAPI specification |
| **REQ-011** | ☑ | As a Requester I want to see the request including the default parameter values what they end up being |
| **REQ-012** | ☑ | As a Requester I want to validate the request against the OpenAPI specification |
| **REQ-013** | ☑ | As a Requester I want to validate the response against the OpenAPI specification |
| **REQ-014** | ☑ | As a Requester I want to see OpenAPI documentation whenever applicable |
| **REQ-015** | ☑ | As a Requester I want to jump to API documentation from hovering over a help link |
| **REQ-016** | ☑ | As a Requester I want to make requests in free form mode (Standard mode) |
| **REQ-017** | ☑ | As a Requester I want to see a preview of the raw HTTP request |
| **REQ-018** | ☑ | As a Requester I want to format the response as JSON, XML, or YAML |
| **REQ-019** | ☑ | As a Requester I want to copy the response to the clipboard |
| **REQ-020** | ☑ | As a Requester I want to save the response to a file |
| **REQ-021** | ☑ | As a Requester I want to print the response |
| **REQ-022** | ☑ | As a Requester I want to be able to filter the response via a JMESPath query |
| **REQ-023** | ☑ | As a Requester I want to be able to highlight the response via a JMESPath query |
| **REQ-024** | ☐ | As a Requester I want to grant API access to individuals or teams without sharing the API key |
| **REQ-025** | ☐ | As a Requester I want to revoke API access to individuals or teams |
