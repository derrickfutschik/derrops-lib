### As a **Administrator**

- [ ] I want to bring my own secret management solution to the use of my APIs.
- [ ] I want to specify the API key to use for the request from the secret management solution.

### As a **Developer**

- [ ] I want to make requests from a Lambda function sitting in my AWS Account
- [ ] I want to make requests from a Lambda function from SLAOps.
