### As a **Administrator**

- [ ] Bring my own secret management solution to the use of my APIs.
- [ ] Specify the API key to use for the request from the secret management solution.

### As a **Developer**

- [ ] Requests from a Lambda function sitting in my AWS Account.
- [ ] Requests from a Lambda function from SLAOps.
- [ ] Local requests from the browser should not be permitted where by sensitive secrets are involved.
- [ ] Support a template based code generation system which is pluggable into the requester, and available on github.
