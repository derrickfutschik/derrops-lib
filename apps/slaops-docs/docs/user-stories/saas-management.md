### As an **Admin**

- [ ] I want to be able to create a new SAAS application and save it in the SQL database
- [ ] I want to be able to edit a SAAS application and save it in the SQL database
- [ ] I want to be able to delete a SAAS application and remove it from the SQL database
- [ ] I want to be able to view a SAAS application and see the details in the SQL database
