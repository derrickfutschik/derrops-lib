## General Functionality

As a Requester

- [x] I want to be able to sign in with AWS Cognito
- [ ] I want to be able to sign out with AWS Cognito
- [ ] I want to be able save secrets in a secure way

## OpenAPI Functionality

As a Requester

- [x] I want to be able to make requests according to the OpenAPI specification.
- [x] I want to be able to see the request including the default parameter values what they end up being
- [x] I want to validate the request against the OpenAPI specification
- [ ] I want to validate the response against the OpenAPI specification
- [x] I want to see OpenAPI documentation whenever applicable
- [ ] I want to be able to jump to API documentation from hovering over a help link

## Request Functionality

As a Requester

- [x] I want to be able to make requests according to the OpenAPI specification.
- [x] I want to be able to make requests in free form mode (Standard mode).
- [x] I want to be able to see a preview of the raw HTTP request

## Response Functionality

As a Requester

- [x] I want to be able to format the response as JSON, XML, or YAML
- [x] I want to be able to copy the response to the clipboard
- [x] I want to be able to save the response to a file
- [x] I want to be able to print the response
- [x] I want to be able to be able to filter the response via a JMESPath query
- [x] I want to be able to be able to highlight the response via a JMESPath query
