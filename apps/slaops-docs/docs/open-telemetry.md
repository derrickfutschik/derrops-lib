# OpenTelemetry

OpenTelemetry is a set of tools, APIs, and SDKs. It is a CNCF project and is the standard for distributed tracing.

It is used to trace the requests and responses of the API.

It is used to trace the latency of the API.

It is used to trace the errors of the API.

It is used to trace the throughput of the API.

It is used to trace the availability of the API.

Segments, and attributes need to be added where possible for integration.
