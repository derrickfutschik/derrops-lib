# Cloud Request User Stories

| ID          | Done | Description                                                                                                                             |
| ----------- | ---- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **CLD-001** | ☐    | As an Administrator I want to Bring my own secret management solution to the use of my APIs                                             |
| **CLD-002** | ☐    | As an Administrator I want to Specify the API key to use for the request from the secret management solution                            |
| **CLD-003** | ☐    | As a Developer I want to make Requests from a Lambda function sitting in my AWS Account                                                 |
| **CLD-004** | ☐    | As a Developer I want to make Requests from a Lambda function from SLAOps                                                               |
| **CLD-005** | ☐    | As a Developer Local requests from the browser should not be permitted where by sensitive secrets are involved                          |
| **CLD-006** | ☐    | As a Developer I want to Support a template based code generation system which is pluggable into the requester, and available on github |
