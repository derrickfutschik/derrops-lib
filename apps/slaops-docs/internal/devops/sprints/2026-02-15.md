# Add OpenAPI Specs

## Half Flow

1. Accept a body containing an OpenAPI Spec
2. Index the OpenAPI spec into OpenSearch
3. Return the OpenAPI spec

## Full Flow

1. Generate Pre-Signed URL for OpenAPI Spec
2. Upload OpenAPI Spec to Staging Bucket
3. Read and Parse OpenAPI Spec
4. Review the OpenAPI Spec
5. Index the OpenAPI Spec into OpenSearch
6. Copy the OpenAPI Spec to the Production Bucket

# Search for an OpenAPI Spec

1. Search for an OpenAPI Spec by domain, path, method, operationId, tag, etc
2. Return the OpenAPI Spec
