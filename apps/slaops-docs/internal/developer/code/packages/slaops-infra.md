# SLAOps Infrastructure

This package contains the infrastructure code for the SLAOps platform. It is used to deploy the SLAOps platform to AWS.
It utilizes the AWS CDK to deploy the infrastructure.
It is a TypeScript package that is used to deploy the SLAOps platform to AWS.
The infrastructure layer is does not support feature deploys unlike the amplify deploy of the slaops-ports app or the lambda deploy of the slaops-cloud app.
