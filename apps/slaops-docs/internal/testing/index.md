# Test Reports

Internal test reports, coverage summaries, and quality metrics.

:::warning Internal only
This section is restricted to SLAOps staff.
:::

## Contents

Add test report files to this directory. Suggested structure:

```
internal/testing/
├── index.md                 ← This file
├── unit/                    ← Unit test coverage reports
├── integration/             ← Integration test results
└── e2e/                     ← End-to-end test results
```
