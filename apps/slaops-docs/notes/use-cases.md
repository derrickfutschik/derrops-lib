# HTTP Logs Supported

| Log Type | Description |
|----------|-------------|
| SAAS Logs | HTTP calls to your own SAAS applications |
| Agentic Logs | HTTP calls from your Agents when they access other HTTP calls |
| API Gateway Logs | API Gateway Access Logs from AWS can be transformed |
| Load Balancer Logs | Access Logs from your Load Balancer |
| Server Logs | Logs at your application level can be added into the platform |
| Web Browser Logs | Logs from Javascript or Http Clients can be added and transformed into the platform |
| Mobile Client Logs | SLAOPs Platform can be used on your own Mobile Client Logs. Using the platform you can generate the view of a consumer of your own Mobile Client |
| Desktop Client Logs | SLAOPs Platform can be used on your own Desktop Client Logs. Using the platform you can generate the view of a consumer of your own Desktop Client |
