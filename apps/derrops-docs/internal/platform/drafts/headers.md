Following defaults for headers, these can be configured.

| Header                   | Description                          |
| ------------------------ | ------------------------------------ |
| DERROPS-Service           | The name of the service              |
| DERROPS-External-Id       | The external id of the request       |
| DERROPS-External-Trace-Id | The external trace id of the request |
| DERROPS-External-Span-Id  | The external span id of the request  |
